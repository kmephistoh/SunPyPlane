## How to Run
  
  请使用python 3.6和pygame 1.9.3 以上版本运行。
  
  git clone https://github.com/sun1218/SunPyPlane.git
  
  python main.py
  


## How to Play

### 移动

- W & UP   : 向上移动
- S & DOWN : 向下移动
- A & LEFT : 向左移动
- D & RIGHT: 向右移动




### 攻击
- J        : 发射子弹
- Space    : 清屏炸弹 （一条命有10个）
- Return   : 切换子弹 （一共有十种，每种的伤害力不一）





### 护盾
- Tab      : 切换护盾模式




### 暂停 & 继续

暂停只会停止移动，但不会停止刷敌机。
